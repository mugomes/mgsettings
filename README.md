# MGSettings

MGSettings é um componente para adicionar e consultar configurações em JSON.

## Recursos

- Carrega o arquivo de configurações
- Armazena configurações
- Consulta configurações
- Salva o arquivo de configurações em formato JSON

## Instalação

`go get github.com/mugomes/mgsettings`

## Information

 - [Page MGSettings](https://github.com/mugomes/mgsettings)
